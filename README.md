DevOps Integration Project
This project implements a complete DevOps pipeline for a Spring Boot application, focusing on automation, monitoring, and artifact management.

Overview
The project covers key DevOps practices:

Continuous Integration and Deployment (CI/CD) with Jenkins
Code quality analysis with SonarQube
Artifact management with Nexus
Application monitoring using Prometheus and Grafana
Deployment with Docker and Docker Compose

Technologies Used
Spring Boot, Maven, Jenkins
Docker, SonarQube, Nexus
Prometheus, Grafana
CI/CD Pipeline

Steps:
Build and test with Maven.
Analyze code quality using SonarQube.
Store build artifacts in Nexus.
Containerize and deploy using Docker.

Tools:
Jenkins automates the pipeline.
Docker Hub stores application images.

Monitoring
Grafana: Tracks application and Jenkins performance.
Prometheus: Collects metrics for real-time monitoring.
